Class ResourceIteratorClassFactory

Factory for creating Guzzle\Service\Resource\ResourceIteratorInterface objects using a convention of storing iterator classes under a root namespace using the name of a Guzzle\Service\Command\CommandInterface object as a convention for determining the name of an iterator class. The command name is converted to CamelCase and Iterator is appended -- (e.g. camel_case => CamelCaseIterator).
