Aggregate multi-valued parameters by joining the values using a comma

Aggregate multi-valued parameters by joining the values using a comma

$queryString = new \Guzzle\Http\QueryString();
$queryString->setAggregator($queryString->aggregateUsingComma);
$queryString->replace(array(
    'value' => array(1, 2, 3)
));
