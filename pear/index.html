Guzzle PEAR channel

Registering the channel: 
pear channel-discover guzzlephp.org/pear
Listing available packages: 
pear remote-list -c guzzle
Installing a package: 
pear install guzzle/package_name
Installing a specific version/stability: 
pear install guzzle/package_name-1.0.0
pear install guzzle/package_name-beta
Receiving updates via a feed: 
http://guzzlephp.org/pear/feed.xml

guzzle - HTTP client and REST client framework

Guzzle is a PHP 5.3+ HTTP client and framework for building RESTful web service clients
